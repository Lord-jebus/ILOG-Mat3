Ecuaciones Diferenciales:
Lineales de Primer Orden

Método del Factor Integrante

Matemáticas 3 UTEC / Ingeniería Logística — Clase 24

Fecha: 12/6/2026

Objetivos de la Sesión

Objetivo General Resolver ecuaciones diferenciales lineales de primer orden mediante el método del factor integrante.

Objetivos Específicos:

  1. Identificar la forma estándar de una ED lineal de primer orden.
  2. Calcular el factor integrante \(\mu(x)\).
  3. Aplicar la propiedad de la derivada de un producto para resolver la ecuación.
  4. Resolver problemas aplicados de mezclas y circuitos simples.

Definición: ED Lineal de Primer Orden

Una ecuación diferencial es lineal de primer orden si puede escribirse en la forma:

Forma Estándar $$\frac{dy}{dx} + P(x)y = Q(x)$$ Donde \(P(x)\) y \(Q(x)\) son funciones continuas de \(x\). \(y\) y \(y'\) aparecen en primer grado.

Ejemplos:

  • \(y' + 2xy = x\) (Lineal: \(P=2x, Q=x\)).
  • \(y' = y\) (Lineal: Se escribe \(y' - y = 0\)).
  • \(y' + y^2 = 0\) (No lineal: tiene \(y^2\)).
Nota: Si \(Q(x) = 0\), se llama homogénea. Si \(Q(x) \neq 0\), es no homogénea.

El "Truco": Factor Integrante

El problema No podemos integrar directamente \(\int (y' + P y) dx\) porque \(y\) es desconocida.
La solución Buscamos multiplicar toda la ecuación por una función \(\mu(x)\) tal que el lado izquierdo se convierta en la derivada de un producto. $$\mu(x)\left(\frac{dy}{dx} + P(x)y\right) = \mu(x) Q(x) \implies \frac{d}{dx}[\mu(x) \cdot y] = \mu(x) Q(x)$$

Si derivamos el producto: \((\mu y)' = \mu y' + \mu' y\). Comparando con \(\mu y' + \mu P y\), necesitamos que \(\mu' = \mu P\).

Fórmula del Factor Integrante Resolviendo \(\mu' = P \mu\), obtenemos: $$\mu(x) = e^{\int P(x)\,dx}$$

Pasos para Resolver (Algoritmo)

  1. Escribir en Forma Estándar: Asegurarse que el coeficiente de \(y'\) sea 1. \(y' + P(x)y = Q(x)\)
  2. Identificar \(P(x)\): (Ignoramos \(Q(x)\) por ahora).
  3. Calcular el Factor Integrante \(\mu(x)\): \(\mu(x) = e^{\int P(x) dx}\) (No sumar constante de integración \(C\) aquí).
  4. Multiplicar toda la ecuación original por \(\mu(x)\):
  5. Reconocer la derivada del producto: El lado izquierdo será \(\frac{d}{dx}[\mu(x) y]\).
  6. Integrar ambos lados: \(\mu(x) y = \int \mu(x) Q(x) dx + C\)
  7. Despejar \(y\).

Ejemplo Guiado 1: Ecuación No Homogénea

Problema Resolver \(y' + 3y = 6x\).
  1. Forma estándar: Ya está. \(P(x) = 3\), \(Q(x) = 6x\).
  2. Factor Integrante: \(\mu(x) = e^{\int 3 dx} = e^{3x}\)
  3. Multiplicar ecuación: \(e^{3x} y' + 3e^{3x} y = 6x e^{3x}\). Lado izquierdo: \(\frac{d}{dx}[e^{3x} y]\).
  4. Integrar: \(e^{3x} y = \int 6x e^{3x} dx\). Usando integración por partes (\(u=6x, dv=e^{3x}dx\)): \(e^{3x} y = 2x e^{3x} - \frac{2}{3}e^{3x} + C\)
  5. Despejar \(y\): \(\boxed{y = 2x - \frac{2}{3} + C e^{-3x}}\)

Ejemplo Guiado 2: Limpieza de un Tanque

Problema Un tanque contiene \(V\) litros. La cantidad de sal \(S(t)\) satisface: \(\frac{dS}{dt} + \frac{r}{V}S = 0\). Resolver para \(S(t)\).
  1. \(P(t) = \frac{r}{V}\) (Constante).
  2. \(\mu(t) = e^{\int \frac{r}{V} dt} = e^{\frac{r}{V}t}\).
  3. Ecuación: \(\frac{d}{dt}[S e^{\frac{r}{V}t}] = 0\).
  4. Integrar: \(S e^{\frac{r}{V}t} = C\).
  5. Solución: \(S(t) = C e^{-\frac{r}{V}t}\)
Interpretación Decaimiento exponencial (la sal se "lava" del tanque).

Práctica Activa

Instrucciones Resuelva las siguientes ED lineales. Tiempo: 30 minutos.
  1. \(\displaystyle x y' + y = x^2 \sin x\) (Sugerencia: Divida por \(x\) primero para normalizar).
  2. \(\displaystyle \frac{dy}{dx} - \frac{2}{x} y = x^3\) (Con condición \(y(1)=0\)).
  3. \(\displaystyle y' + y = e^x\) (Sugerencia: El factor integrante es \(e^x\)).
  4. Aplicación: Ley de Enfriamiento de Newton: \(\frac{dT}{dt} + kT = k T_m\). Resuelva para \(T(t)\).

Resolución Paso a Paso — Ejercicio 1

Enunciado \(x y' + y = x^2 \sin x\)
  1. Paso 1: Normalizar. Dividir por \(x\): \(y' + \frac{1}{x}y = x \sin x\)
  2. Paso 2: Factor Integrante. \(P = 1/x \implies \mu = e^{\ln x} = x\)
  3. Paso 3: Multiplicar y reconocer derivada. \(x y' + y = x^2 \sin x \implies \frac{d}{dx}[xy] = x^2 \sin x\)
  4. Paso 4: Integrar (2 veces por partes). \(xy = \int x^2 \sin x dx = -x^2\cos x + 2x\sin x + 2\cos x + C\)
Resultado Final: \(y(x) = -x\cos x + 2\sin x + \frac{2}{x}\cos x + \frac{C}{x}\)

Resolución Paso a Paso — Ejercicio 2 (PIV)

Enunciado \(\frac{dy}{dx} - \frac{2}{x} y = x^3\), con \(y(1)=0\)
  1. Paso 1: Factor Integrante. \(P = -2/x \implies \mu = e^{-2\ln x} = x^{-2}\)
  2. Paso 2: Multiplicar. \(\frac{d}{dx}[x^{-2}y] = x^{-2} \cdot x^3 = x\)
  3. Paso 3: Integrar. \(x^{-2}y = \frac{x^2}{2} + C \implies y = \frac{x^4}{2} + Cx^2\)
  4. Paso 4: Aplicar PVI \(y(1)=0\). \(0 = \frac{1}{2} + C \implies C = -1/2\)
Resultado Final: \(y(x) = \frac{x^4 - x^2}{2}\)

Resolución Paso a Paso — Ejercicio 3

Enunciado \(y' + y = e^x\)
  1. Paso 1: Factor Integrante. \(P=1 \implies \mu = e^x\)
  2. Paso 2: Multiplicar. \(e^x y' + e^x y = e^{2x} \implies \frac{d}{dx}[e^x y] = e^{2x}\)
  3. Paso 3: Integrar. \(e^x y = \frac{1}{2}e^{2x} + C\)
  4. Paso 4: Despejar. \(y = \frac{1}{2}e^x + Ce^{-x}\)
Resultado Final: \(y(x) = \frac{1}{2}e^x + Ce^{-x}\)

Resolución Paso a Paso — Ejercicio 4 (Aplicación)

Enunciado \(\frac{dT}{dt} + kT = k T_m\)
  1. Paso 1: Factor Integrante. \(P = k \implies \mu = e^{kt}\)
  2. Paso 2: Multiplicar. \(\frac{d}{dt}[T e^{kt}] = k T_m e^{kt}\)
  3. Paso 3: Integrar. \(T e^{kt} = T_m e^{kt} + C\)
  4. Paso 4: Despejar. \(T(t) = T_m + Ce^{-kt}\)
Interpretación La temperatura converge exponencialmente hacia la temperatura ambiente \(T_m\).

Caso de Estudio Industrial (Logística)

Flujo de Inventario con Demanda Dependiente del Tiempo

Contexto: Entradas a tasa constante \(R\), demanda creciente \(D(t) = a + bt\). Inventario \(I(t)\): \(\frac{dI}{dt} = R - (a + bt)\).

Problema Si hay deterioro proporcional al inventario \(kI\): \(\frac{dI}{dt} + kI = R - a - bt\). Encontrar \(I(t)\) y predecir quiebre de stock.

Solución (Factor Integrante):

  1. \(P(t)=k, Q(t)=R-a-bt\). Factor: \(e^{kt}\).
  2. \(I(t) = e^{-kt} \left[ I_0 + \int_0^t e^{k\tau}(R - a - b\tau) d\tau \right]\)
  3. Resolver por partes permite predecir evolución del stock considerando deterioro y demanda creciente.

Código MATLAB - Solver Simbólico

Verificación automática de soluciones.
% Ejercicio 3: y' + y = exp(x)
syms y(x)
ode = diff(y, x) + y == exp(x);
sol = dsolve(ode);
disp('Solucion General:');
pretty(sol)

% Verificar una condicion inicial y(0) = 1
cond = y(0) == 1;
sol_pvi = dsolve(ode, cond);
disp('Solucion Particular y(0)=1:');
pretty(sol_pvi)
Uso: Copiar en MATLAB. `dsolve` devuelve la solución simbólica exacta para validar cálculos manuales.
