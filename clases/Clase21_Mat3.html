Objetivos de la Sesión

Objetivo General Comprender los conceptos fundamentales de las ecuaciones diferenciales ordinarias (EDO), incluyendo su clasificación y tipos de soluciones.

Objetivos Específicos:

  1. Definir qué es una Ecuación Diferencial Ordinaria (EDO).
  2. Identificar el orden y el grado de una ecuación.
  3. Distinguir entre solución general, solución particular y solución singular.
  4. Verificar si una función dada es solución de una EDO.

Introducción: El Lenguaje del Cambio

Problema algebraico: Encontrar un número \(x\) tal que \(x^2 - 4 = 0\). Solución: \(x = 2\). (Valor estático).

Problema diferencial: Encontrar una función \(y(x)\) tal que su derivada sea igual a la función misma.

$$\frac{dy}{dx} = y$$

Pregunta: ¿Qué función conocida es igual a su propia pendiente?

Definición de Ecuación Diferencial Es una ecuación que relaciona una función desconocida (variable dependiente) con sus derivadas respecto a una o más variables independientes. $$F(x, y, y', y'', \dots, y^{(n)}) = 0$$

Aplicación: Modelan fenómenos dinámicos: crecimiento poblacional, decaimiento radioactivo, enfriamiento, movimiento, circuitos eléctricos.

Clasificación de Ecuaciones Diferenciales

1. Por su Orden: Es el orden de la derivada más alta que aparece en la ecuación.

  • Primer Orden: \(\frac{dy}{dx} + y = e^x\)
  • Segundo Orden: \(y'' + 4y' = 0\)
  • \(n\)-ésimo Orden: \(y^{(n)} + \dots = 0\)

2. Por su Linealidad:

Una ecuación es lineal si cumple:
  1. La variable dependiente \(y\) y sus derivadas están elevadas a la primera potencia.
  2. No hay productos entre \(y\) y sus derivadas (ej. \(y \cdot y'\)).
  3. No hay funciones trascendentes de \(y\) (ej. \(\sin(y)\), \(e^y\)).

Ejemplos:

  • Lineal: \(y'' + x^2 y = \sin x\).
  • No Lineal: \(y' + y^2 = 0\) (potencia 2) o \(y y' = 1\) (producto).

Tipos de Soluciones

Resolver una EDO significa encontrar la función \(y(x)\) que satisface la ecuación.

Solución General Es una solución que contiene tantas constantes arbitrarias independientes como sea el orden de la ecuación.
  • Ecuación de 1er orden \(\to\) 1 constante (\(C\)).
  • Ecuación de 2do orden \(\to\) 2 constantes (\(C_1, C_2\)).

Representa una familia de curvas.

Solución Particular Es una solución obtenida al dar valores específicos a las constantes (\(C\)), generalmente mediante condiciones iniciales. Representa una curva específica de la familia.
Problema de Valor Inicial (PVI) Ecuación Diferencial + Condición Inicial (\(y(t_0) = y_0\)).

Ejemplo Guiado 1: Verificación de Solución

Problema Demostrar que \(y = x^2 + C e^x\) es la solución general de la ecuación: $$y' - y = 2x - x^2$$

Solución (Método de Verificación):

  1. Calcular la derivada de \(y\):
    \(y = x^2 + C e^x \implies y' = 2x + C e^x\)
  2. Sustituir \(y\) y \(y'\) en la ecuación diferencial: $$(2x + C e^x) - (x^2 + C e^x) = 2x - x^2$$
  3. Simplificar: $$2x + \cancel{C e^x} - x^2 - \cancel{C e^x} = 2x - x^2$$ $$2x - x^2 = 2x - x^2 \quad \text{(Identidad verdadera)}$$

Conclusión: La función propuesta es solución de la EDO.

Ejemplo Guiado 2: Problema de Valor Inicial

Problema Encontrar la solución particular de \(y' = 3x^2\) que satisface \(y(1) = -1\).
  1. Resolver la ED (Integrar): \(y = \int 3x^2 dx = x^3 + C\)
  2. Aplicar Condición Inicial: Sabemos que cuando \(x=1\), \(y=-1\).
    \(-1 = (1)^3 + C\)
  3. Despejar \(C\): \(C = -1 - 1 = -2\)
  4. Escribir Solución Particular: \(\boxed{y = x^3 - 2}\)
Interpretación De la familia de curvas \(y=x^3+C\) (desplazamiento vertical), seleccionamos la que pasa por el punto \((1, -1)\).

Instrucciones Clasifique y verifique. Tiempo: 30 minutos.

Soluciones a los Ejercicios (Resumen)

Ejercicio 1: \(y'' + y = 0\): Orden 2, Lineal. \quad \(y' + y^2 = \sin x\): Orden 1, No Lineal. \quad \((y')^3 + y = x\): Orden 1, No Lineal.
Ejercicio 2: \(y = 2e^{3x} \to y' = 6e^{3x} = 3(2e^{3x}) = 3y\). Satisface \(y' - 3y = 0\). Sí es solución.
Ejercicio 3: \(y = -2x^2 + C\). Con \(y(0)=5 \to 5 = 0 + C \implies C=5\). Solución: \(y = -2x^2 + 5\).
Ejercicio 4: EDO: \(\frac{dT}{dt} = -k(T - T_m)\), donde \(k>0\) es constante de enfriamiento y \(T_m\) es temperatura ambiente.

Resolución Paso a Paso — Ejercicio 1

  1. \(y'' + y = 0\): La derivada más alta es \(y''\) (2ª orden). \(y\) aparece a la potencia 1 y sin productos/funciones trascendentes. \(\to\) Orden 2, Lineal.
  2. \(\frac{dy}{dx} + y^2 = \sin x\): Derivada más alta: \(y'\) (1er orden). Contiene \(y^2\) (potencia 2). \(\to\) Orden 1, No Lineal.
  3. \(\left(\frac{dy}{dx}\right)^3 + y = x\): Derivada más alta: \(y'\) (1er orden). La derivada está elevada al cubo. \(\to\) Orden 1, No Lineal.
Regla de oro: La linealidad depende exclusivamente de cómo aparece \(y\) y sus derivadas, no de \(x\) ni de funciones de \(x\).

Resolución Paso a Paso — Ejercicio 2

Verificar si \(y = 2e^{3x}\) satisface \(y' - 3y = 0\)
  1. Paso 1: Derivar la función propuesta.
    \(y = 2e^{3x} \implies y' = \frac{d}{dx}(2e^{3x}) = 2 \cdot 3e^{3x} = 6e^{3x}\)
  2. Paso 2: Sustituir \(y\) y \(y'\) en la EDO.
    \(y' - 3y = 6e^{3x} - 3(2e^{3x})\)
  3. Paso 3: Simplificar.
    \(6e^{3x} - 6e^{3x} = 0\)
Resultado: La igualdad \(0=0\) se cumple para todo \(x\). Por lo tanto, sí es solución.

Resolución Paso a Paso — Ejercicio 3

PVI: \(y' = -4x\), condición \(y(0)=5\)
  1. Paso 1: Integrar para hallar la solución general.
    \(y(x) = \int -4x \, dx = -2x^2 + C\)
  2. Paso 2: Aplicar la condición inicial \(y(0)=5\).
    Sustituir \(x=0, y=5\) en la general: \(5 = -2(0)^2 + C\)
  3. Paso 3: Despejar \(C\).
    \(5 = 0 + C \implies C = 5\)
Solución Particular Final: $$y(x) = -2x^2 + 5$$

Resolución Paso a Paso — Ejercicio 4

Modelar la Ley de Enfriamiento de Newton
  1. Identificar variables: Temperatura del objeto: \(T(t)\). Temperatura ambiente: \(T_m\) (constante).
  2. Traducir "tasa de cambio": \(\frac{dT}{dt}\)
  3. Traducir "proporcional a la diferencia": \(\propto (T - T_m)\)
  4. Introducir constante de proporcionalidad \(k\): Como el objeto se enfría si \(T > T_m\), la tasa debe ser negativa. Por eso se usa \(-k\) con \(k>0\).
Ecuación Diferencial Resultante: $$\frac{dT}{dt} = -k(T - T_m), \quad k > 0$$ (Esta es una EDO lineal de primer orden con variables separables).

Caso de Estudio Industrial (Logística)

Título: Ajuste de Modelos de Inventario Dinámico

Contexto: Un almacén gestiona un producto perecedero. La tasa de cambio del inventario \(I(t)\) es constante por salida fija, pero hay incertidumbre en la medición inicial.

Modelo Matemático: $$\frac{dI}{dt} = -50 \text{ unidades/día}$$

Problema: Contador dice 200 uds en día 0. Sistema de seguridad marca 0 uds el día 4. ¿Era correcta la medición inicial? ¿Cuál era el inventario real en \(t=0\)?

  1. Integrar: \(I(t) = \int -50 \, dt = -50t + C\)
  2. Usar dato real: \(I(4) = 0 \implies 0 = -50(4) + C \implies C = 200\)
  3. Conclusión: \(I(0) = -50(0) + 200 = 200\).
Veredicto: La medición del contador era correcta. Si el sistema marcara 0 uds el día 3, entonces \(C=150\), indicando un robo o error de 50 uds iniciales.

Código MATLAB - Verificación Simbólica

Uso de MATLAB para verificar soluciones automáticamente.
% Verificación de EDO con Toolbox Simbólico
syms x y(x) C

% 1. Definir la ecuacion diferencial
% Ejemplo: y' - 3y = 0
ode = diff(y, x) - 3*y == 0;

% 2. Proponer una solucion
sol_propuesta = 2*exp(3*x);

% 3. Sustituir y verificar
% Reemplazamos y por la solucion en el lado izquierdo
LHS = subs(lhs(ode), y, sol_propuesta);
RHS = rhs(ode);

disp('Es solucion?');
disp(LHS == RHS); % Debe devolver 1 (True)
Tip: Usar dsolve resuelve automáticamente: dsolve(ode) devuelve C1*exp(3*x).
