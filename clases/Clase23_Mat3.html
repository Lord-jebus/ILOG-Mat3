Método de Separación de Variables

Matemáticas 3 UTEC — Ingeniería Logística — Clase 23

Fecha: 10 de Junio de 2026

Objetivos de la Sesión

Objetivo General Aplicar el método de separación de variables para resolver ecuaciones diferenciales de primer orden.

Objetivos Específicos:

  1. Identificar la forma estándar de una ecuación diferencial de variables separables.
  2. Dominar la manipulación algebraica para separar las variables \(x\) e \(y\).
  3. Integrar ambos lados de la ecuación para encontrar la solución general.
  4. Resolver problemas de valor inicial aplicando condiciones.

¿Qué es una Ecuación de Variables Separables?

Es una EDO de primer orden que puede escribirse en la forma:

Forma Estándar $$\frac{dy}{dx} = g(x) \, h(y)$$ O equivalentemente: $$M(x)\,dx = N(y)\,dy$$

La clave:

Podemos "separar" todo lo que depende de \(x\) (incluyendo \(dx\)) a un lado de la ecuación, y todo lo que depende de \(y\) (incluyendo \(dy\)) al otro lado.

Ejemplos:

  • Sí es separable: \(\frac{dy}{dx} = x y^2 \quad\to\quad \frac{dy}{y^2} = x\,dx\).
  • No es separable: \(\frac{dy}{dx} = x + y\) (No es producto de funciones de \(x\) e \(y\)).

Pasos para Resolver (Método)

  1. Identificar: Verificar si la ecuación es separable.
  2. Separar: Mover algebraicamente los términos.
    $$\frac{dy}{dx} = g(x)h(y) \implies \frac{1}{h(y)}\,dy = g(x)\,dx$$ Nota: Hay que tener cuidado si \(h(y)=0\) (soluciones singulares).
  3. Integrar: Aplicar integral a ambos lados.
    $$\int \frac{1}{h(y)}\,dy = \int g(x)\,dx$$
  4. Resolver: Calcular las integrales y despejar \(y\) (si es posible).
  5. Constante: Agregar la constante de integración \(C\) (solo una vez, usualmente en el lado de \(x\) o \(y\)).
Resultado La solución suele ser implícita: \(H(y) = G(x) + C\). A veces se puede despejar \(y\) explícitamente.

Ejemplo Guiado 1: Separación Básica

Problema Resolver la ecuación diferencial \(\displaystyle \frac{dy}{dx} = \frac{x^2}{y}\).

Solución Paso a Paso:

  1. Separar variables: Multiplicamos por \(y\) y por \(dx\).
    \(y\,dy = x^2\,dx\)
  2. Integrar ambos lados: \(\displaystyle \int y\,dy = \int x^2\,dx\)
  3. Calcular integrales: \(\displaystyle \frac{y^2}{2} = \frac{x^3}{3} + C_1\)
  4. Simplificar (Solución Implícita): Multiplicamos por 6 para limpiar fracciones:
    \(3y^2 = 2x^3 + C\) (donde \(C = 6C_1\))
  5. Solución Explícita: \(\displaystyle \boxed{y = \pm \sqrt{\frac{2x^3 + C}{3}}}\)

Ejemplo Guiado 2: Problema de Valor Inicial (PIV)

Problema Resolver \(\displaystyle \frac{dy}{dx} = y \cos x\), sujeto a \(y(0) = 2\).
  1. Separar: \(\displaystyle \frac{dy}{y} = \cos x\,dx \quad (y \neq 0)\)
  2. Integrar: \(\displaystyle \int \frac{1}{y}dy = \int \cos x\,dx \implies \ln|y| = \sin x + C\)
  3. Despejar \(y\) (Exponencial):
    \(|y| = e^{\sin x + C} = e^C e^{\sin x} \implies y = \pm e^C e^{\sin x}\)
    Llamemos \(A = \pm e^C\) (nueva constante). \(\implies y = A e^{\sin x}\)
  4. Aplicar Condición Inicial (\(x=0, y=2\)):
    \(2 = A e^{\sin 0} \implies 2 = A e^0 \implies A = 2\)
  5. Solución Particular: \(\displaystyle \boxed{y = 2 e^{\sin x}}\)

Práctica Activa

Instrucciones Resuelva las siguientes ecuaciones. Tiempo: 30 minutos.
  1. \(\displaystyle \frac{dy}{dx} = e^{x-y}\) (Sugerencia: Recuerde \(e^{x-y} = e^x e^{-y}\)).
  2. \(\displaystyle (x^2 + 1) y' = xy\) (Sugerencia: Separe la \(x\) de la \(y\)).
  3. PIV: \(\displaystyle \frac{dy}{dt} = 2y(1-t)\), con \(y(0) = 1\).
  4. Aplicación: La tasa de crecimiento de una bacteria es proporcional a su población actual. Si la población se triplica en 2 horas, ¿cuál es la ecuación que modela la población \(P(t)\)? (Plantee y encuentre la constante de proporcionalidad).

Resolución Paso a Paso — Ejercicio 1

Enunciado \(\frac{dy}{dx} = e^{x-y}\)
  1. Paso 1: Reescribir usando propiedades de potencias.
    \(e^{x-y} = e^x \cdot e^{-y} \implies \frac{dy}{dx} = e^x e^{-y}\)
  2. Paso 2: Separar variables. Multiplicamos por \(e^y dx\):
    \(e^y \, dy = e^x \, dx\)
  3. Paso 3: Integrar ambos lados.
    \(\int e^y \, dy = \int e^x \, dx \implies e^y = e^x + C\)
  4. Paso 4: Despejar \(y\) (Solución Explícita).
    \(y = \ln(e^x + C)\)
Resultado Final: \(y(x) = \ln(e^x + C)\)

Resolución Paso a Paso — Ejercicio 2

Enunciado \((x^2 + 1) y' = xy\)
  1. Paso 1: Separar variables. \(\frac{dy}{y} = \frac{x}{x^2+1} dx\)
  2. Paso 2: Integrar ambos lados.
    \(\int \frac{1}{y} dy = \int \frac{x}{x^2+1} dx\)
  3. Paso 3: Resolver la integral de la derecha (sustitución \(u = x^2+1\), \(du = 2x\,dx\)).
    \(\ln|y| = \frac{1}{2}\ln(x^2+1) + C\)
  4. Paso 4: Aplicar exponencial y simplificar.
    \(|y| = e^C (x^2+1)^{1/2}\). Llamando \(A = \pm e^C\):
    \(y = A\sqrt{x^2+1}\)
Resultado Final: \(y(x) = A\sqrt{x^2+1}\)

Resolución Paso a Paso — Ejercicio 3 (PIV)

Enunciado \(\frac{dy}{dt} = 2y(1-t)\), con \(y(0) = 1\)
  1. Paso 1: Separar variables. \(\frac{dy}{y} = 2(1-t)\,dt\)
  2. Paso 2: Integrar.
    \(\ln y = 2t - t^2 + C\) (Asumimos \(y>0\) por la condición inicial)
  3. Paso 3: Aplicar condición inicial \(y(0)=1\).
    \(\ln(1) = 0 - 0 + C \implies C = 0\)
  4. Paso 4: Despejar \(y\).
    \(\ln y = 2t - t^2 \implies y = e^{2t - t^2}\)
Resultado Final: \(y(t) = e^{2t - t^2}\)

Resolución Paso a Paso — Ejercicio 4 (Aplicación)

Enunciado Tasa de crecimiento \(\propto\) población actual. Se triplica en 2 horas. Hallar \(k\) y el modelo.
  1. Paso 1: Plantear la EDO.
    \(\frac{dP}{dt} = kP\)
  2. Paso 2: Separar e integrar.
    \(\int \frac{dP}{P} = \int k\,dt \implies \ln P = kt + C \implies P(t) = P_0 e^{kt}\)
  3. Paso 3: Usar dato de crecimiento.
    \(P(2) = 3P_0 \implies 3P_0 = P_0 e^{2k} \implies 3 = e^{2k}\)
  4. Paso 4: Despejar \(k\).
    \(\ln 3 = 2k \implies k = \frac{\ln 3}{2} \approx 0.549 \, \)
Modelo Final: \(P(t) = P_0 e^{\left(\frac{\ln 3}{2}\right)t}\)

Caso de Estudio Industrial (Logística)

Título: Modelo de Difusión de Innovación en Supply Chain

Contexto: Se implementa un nuevo software de gestión. La tasa de adopción es proporcional al número que ya lo usan (\(A\)) y al que aún no (\(N-A\)), donde \(N\) es el total.

Modelo: $$\frac{dA}{dt} = k A (N - A)$$

Problema: \(N=100\), \(A(0)=5\), \(A(2)=30\). ¿Cuántos lo habrán adoptado en la semana 4?

  1. Separar: \(\frac{dA}{A(N-A)} = k\,dt\)
  2. Fracciones parciales: \(\frac{1}{N}\left(\frac{1}{A} + \frac{1}{N-A}\right)dA = k\,dt\)
  3. Integrar: \(\ln\left(\frac{A}{N-A}\right) = Nkt + C\)
  4. Despejar: \(A(t) = \frac{N}{1 + C e^{-Nkt}}\)
  5. Con datos: Encontrar \(k\) y \(C\) usando \(A(0)=5\) y \(A(2)=30\). (Cálculo numérico).
  6. Predecir \(A(4)\).
Resultado: Modelo logístico clásico. Predice la saturación del mercado.

Código MATLAB - Resolución Simbólica

Uso de dsolve para verificar soluciones.
% Ejercicio 1: dy/dx = e^(x-y)
syms y(t) x
% dsolve usa t por defecto como variable independiente
ode = diff(y, t) == exp(t - y);

% Resolver
sol = dsolve(ode);
disp('Solucion General:');
disp(sol);

% Verificar con PVI (Ejemplo 2: y' = y*cos(x), y(0)=2)
syms y(x)
ode2 = diff(y, x) == y * cos(x);
cond = y(0) == 2;
sol2 = dsolve(ode2, cond);
disp('Solucion Particular Ejemplo 2:');
disp(sol2);
Tip: dsolve resuelve automáticamente y devuelve la solución simbólica. Útil para comprobar cálculos manuales.

Cierre Conceptual

Resumen de la Clase:

  • El método de separación de variables es la "primera línea de ataque" para una EDO.
  • Requiere que la ecuación sea un producto de una función de \(x\) por una función de \(y\).
  • El proceso es puramente algebraico (separar) y calculístico (integrar).
